作業系統需求：
      Windows 7(含)以上。

軟體功能：
(一) 自動偵測檔案傳入，當有檔案傳入時執行以下工作：
      1. 將檔案移動到工作目錄(安裝路徑\dataPool)。
      2. 執行安裝路徑下的 batch 檔(安裝路徑\earthquake.bat)，用戶可自由編寫此檔案。
(二) 在 log 資料夾(安裝路徑\eventlog)產生事件紀錄，檔案名稱為 "檔名+上游名稱.log"。
(三) 當傳入相同的檔案時，後到的檔案不處理，並在 log 中留下記錄。
(四) 傳入過期檔案時不處理，並在 log 中留下記錄。

注意事項：
(一) 卸載時會將安裝的所有檔案移除。
(二) 軟體更新時，若無更改安裝路徑不需先卸載，直接安裝覆蓋即可。
